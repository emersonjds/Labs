Operadores bit a bit lógicos

Conceitualmente, os operadores bit a bit lógicos funcionam da seguinte maneira:

  1. Os operandos são convertidos em inteiros de 32 bits e expressos como uma série de bits (zeros e uns).

  2. Cada bit do primeiro operando é pareado com o bit correspondente do segundo operando: primeiro bit com primeiro bit, segundo bit com segundo bit e assim por diante.

  3. O operador é aplicado a cada par de bits e o resultado é construído bit a bit.
